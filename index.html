False Recall Experiment

A research project designed to study the cognitive mechanisms behind false memory formation. The experiment analyzes user recall data and provides insights into the accuracy of memory recall.

Problem: Memory distortion affects decision-making and eyewitness reliability. Understanding the mechanisms of false recall can improve interventions for memory-related issues.

Solution: Developed and replicated an interactive experiment interface using JavaScript and R to collect and analyze user data. The system uses statistical models to measure memory accuracy and visualize results.

Don't Get It Twisted

A fun, real-time live competition/duel App through tongue twisters to help users improve their pronunciation skills.

Problem: When it comes to the complexities of language, such as similar words/phrases, people view it as an obstacle they must overcome in daily conversations. Existing language learning apps lack the engagement and real-time interaction necessary for effective skill development.

Why would you use this app? Don't Get It Twisted enhances speech therapy by tailoring exercises for individual language improvement, such as articulation and diction, offering timely feedback for noticeable progress. Additionally, users can socialize and network through live competitions, promoting connections for enhanced language learning. Improved pronunciation boosts confidence in social interactions, facilitating effective communication in various contexts.
